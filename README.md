# Elogroup Challenge

## Getting Started

Clone the project

```sh
git clone https://github.com/lucaswilliamgomes/leads.git
```

## Install dependencies

```sh
npm install
```

## Running the project

```sh
npm start
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
